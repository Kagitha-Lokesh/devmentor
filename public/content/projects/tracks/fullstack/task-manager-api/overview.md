# Task Manager REST API

## What You Will Build

A full-stack **Task Management application** comprised of:
1. A **Spring Boot backend** providing a RESTful JSON API, backed by an H2 or MySQL database using Hibernate / Spring Data JPA.
2. A **React frontend** built with Vite, consuming the API through Axios calls.

---

## Why This Project Matters

This project teaches you how to construct standard enterprise web apps:
- Design **RESTful interfaces** conforming to HTTP standards (verbs, response status codes, JSON models).
- Configure cross-origin resource sharing (**CORS**).
- Structure database tables using **JPA ORM mapping annotations**.
- Implement validation constraints and handle errors globally.
- Consume remote APIs asynchronously inside React hooks.
