# Personal Developer Portfolio Website

## What You Will Build

A fully responsive, animated personal portfolio website built with **React** and **Vite** — the same stack used by thousands of production web applications today.

This isn't a tutorial. This is a real deliverable.

By the end of this project, you will have a live, publicly deployed website that represents you professionally to recruiters, clients, and collaborators worldwide.

---

## Why This Project Matters

Your portfolio is the **first thing every employer sees**.

Before your CV. Before your interview.

A well-engineered portfolio demonstrates:
- You can **build real UIs** with modern tooling.
- You understand **component architecture**.
- You care about **accessibility and performance**.
- You can **deploy** and maintain production-grade sites.

---

## What You Will Build Specifically

| Section | Description |
|---|---|
| **Navigation** | Sticky navbar with mobile hamburger menu and active route highlighting |
| **Hero** | Fullscreen hero with name, title, CTA button, and social links |
| **About** | Two-column layout with photo, bio, and stat counters |
| **Skills** | Responsive grid of technology cards with proficiency indicators |
| **Projects** | Card grid with technology tags, GitHub, and demo links |
| **Contact** | Validated form with success state |
| **Footer** | Social links and copyright |

---

## Technology Stack

| Technology | Purpose |
|---|---|
| React 18 | UI component library |
| Vite | Build tool and dev server |
| React Router v6 | Client-side routing |
| CSS3 (Vanilla) | Styling — no Tailwind or UI libraries |
| Intersection Observer API | Scroll animations |
| GitHub Actions | CI/CD pipeline |
| GitHub Pages | Free static hosting |
