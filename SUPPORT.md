# Support Guide

If you run into issues, please use the following resources:

## 1. Documentation
Always consult the [Architecture Guide](./docs/Architecture.md) and [Developer setup Guide](./docs/DeveloperGuide.md) first.

## 2. Issues & Forums
Search through active issues on the GitHub repository. If you find a new bug, please open a bug report issue using the provided issue templates.

## 3. Help Contact
For other support queries, write to support@javamentor.ai.
