# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0-RC3] — 2026-07-10
### Added
- GitHub Actions CI build check config pipeline (`main.yml`).
- Architectural manuals and specifications suites.
- Open-source repository compliance files (README, contributing rules, licenses).

## [1.0.0-RC2] — 2026-07-10
### Added
- Standardized UI component primitive catalog inside `/components/ui/`.
- Search registry generation builder outputs.
- Playwright E2E browser automation scripts.
- Vitest unit tests configurations.
- Monaco Code Editor lazy-loading Suspense wrapper.
- Incremental list rendering pagination inside Learning Timeline.

## [1.0.0-RC1] — 2026-07-09
### Added
- User-isolated background sync queues (`pending_mutations_${uid}`).
- Composite key prefixes for notes, bookmarks, timelines cached records.
- Global authentication state scrubbing mechanism on logout.
- DOMPurify HTML element sanitizers.
