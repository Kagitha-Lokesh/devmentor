# DevMentor — Java Full Stack Learning Platform

DevMentor is a high-fidelity learning platform helping developers master core Java concepts, compile code, take quizzes, run simulated mock interviews, and manage memory decay curves using active recall.

## Stack
- React JS + Vite
- Tailwind CSS
- Zustand (State Management)
- Monaco Editor (Practice workspace)
- Firebase Auth & Storage

## Running Locally
```bash
# Go to project folder
cd devmentor-ai

# Install dependencies
npm install

# Run Vite local server
npm run dev
```
